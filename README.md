# HANGGARIN

## Description
HANGGARIN is a web-based system designed to help students plan and achieve their goals or career aspirations. It provides guidance, tracking, and resources for personal and academic development.

## Features
- Goal setting and tracking system
- User profile management
- Progress monitoring
- Career and skills recommendations
- User-friendly interface

## Purpose
This system was developed to help students have a clear direction for their future and to monitor their progress effectively.

## Technologies Used
- Python (Django Framework)
- HTML, CSS, JavaScript
- SQLite / MySQL
